# PRD: Personal Profile Page (Interview Style)

## 1. 제품 개요

**프로젝트명:** Interview-Style Personal Profile Page
**프로필 주인공:** Selnet.ai CEO & Co-Founder
**목적:** LinkedIn과 유사하지만, 인터뷰(Q&A) 형식으로 자신의 이력과 이야기를 전달하는 개인 프로필 웹페이지

### 왜 인터뷰 형식인가?
- 일반 이력서/LinkedIn은 나열식(bullet point)으로 정보만 전달
- 인터뷰 형식은 **이야기(narrative)** 로 전달하여 인격과 맥락이 드러남
- 방문자가 마치 대화를 읽듯 자연스럽게 프로필을 탐색할 수 있음
- **기술 × 금융 × 창업**이라는 독특한 커리어 아크를 스토리로 전달하기에 최적

### 프로필 요약
- **현재:** Selnet.ai CEO & Co-Founder — AI 에이전트를 위한 신뢰 웹 인프라 구축
- **학력:** 연세대학교 전기전자공학 학사 → Georgia Tech Management of Technology 석사
- **커리어 아크:** HP(기술PM) → Ilshin Ventures(VC) → KEB하나은행(IB) → 미래에셋(헤지펀드, AUM $1.5B) → BC Labs 창업(블록체인 핀테크/Coinvestor) → Selnet.ai(AI Agent 인프라)

## 3. 페이지 구성 (섹션별 인터뷰 질문)

### 3.1 Hero / 첫인상 섹션
- 프로필 사진 (또는 아바타)
- 이름 + Tagline: "Building Trusted Web Infrastructure for AI Agents"
- 서브타이틀 로테이션: AI Agent Infrastructure / Browser Automation / Blockchain & DePIN / Fintech
- 현재 역할: CEO & Co-Founder @ Selnet.ai
- SNS 링크 (GitHub, LinkedIn, Twitter, Email)

### 3.2 Section 1: "자기소개" (2 Q&A)
- 누구인지 + 핵심 정체성
- 한 문장으로 현재 하는 일 설명 → 에이전트의 웹 활동 장벽(CAPTCHA, UI)을 허무는 신뢰 인프라

### 3.3 Section 2: "AI Agent & Browser Automation" (4 Q&A) ★ 핵심 섹션
- AI 에이전트 인프라에 집중하는 이유
- Browser Automation이 핵심인 이유 (API 없는 웹 99%, visual understanding, anti-detection)
- AI 에이전트 시장 현황 (OpenAI Operator, Anthropic Computer Use, Google Mariner)
- Selnet.ai가 구체적으로 해결하는 3가지 문제: Agent Identity / Trusted Browser Session / Agent Interop Protocol

### 3.4 Section 3: "블록체인 & DePIN" (4 Q&A) ★ 핵심 섹션
- 블록체인 경험 → AI 에이전트 인프라 연결 (trustless verification → agent trust)
- DePIN 관심 계기 (분산 브라우저 노드 네트워크 + 토큰 인센티브)
- 블록체인×AI 교차점 트렌드: Agent-to-Agent Payment / Verifiable Compute(ZKP) / DePIN 브라우저 네트워크
- Coinvestor에서 배운 교훈 (데이터 수요 폭증 → 에이전트 필요성 직감)

## 4. 디자인 요구사항

### 4.1 레이아웃
- **싱글 페이지 스크롤** (SPA)
- 각 인터뷰 섹션이 카드 또는 대화 블록으로 구분
- Q&A가 시각적으로 구분됨 (질문: bold/다른 색상, 답변: 본문)

### 4.2 인터랙션
- 스크롤에 따라 Q&A가 순차적으로 나타나는 애니메이션 (fade-in / slide-up)
- 사이드 네비게이션 또는 상단 탭으로 섹션 간 빠른 이동
- 다크모드 / 라이트모드 토글

### 4.3 비주얼 스타일
- 미니멀하고 깔끔한 디자인
- 인터뷰 느낌을 주는 타이포그래피 (질문은 세리프/이탤릭, 답변은 산세리프 등)
- 프로필 사진, 프로젝트 스크린샷 등 이미지 활용

### 4.4 반응형
- 모바일 / 태블릿 / 데스크탑 완벽 대응
- 모바일에서는 대화 메신저 느낌의 UI도 고려
